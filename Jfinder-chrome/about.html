SYNOPSYS JFINDER

About

Allows detection and minimization of duplicate or similar issues and in turn by reducing total number of issues allows your team to be more productive. It doesn't create it's own index and uses Jira's own free text index for search. If has various performance optimizations. For example, it asynchronously fetches similar issues and if you don't keep "Similar Issues Tab" open, it never fetches similar issues on issue page..
